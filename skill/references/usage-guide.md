# 📖 weight-loss-skill 快速上手指南

## 开箱三步

### 第一步：建档

```
减脂管家，帮我建档
身高：175cm
体重：75kg
目标：70kg
年龄：25岁
性别：男
口味：清淡
预算：¥50/天
就餐：公司食堂 + 外卖
忌口：无
运动：跑步 + 力量
```

→ AI 会自动算 BMR/TDEE/建议摄入，存入 Memory。

### 第二步：每日使用

| 场景 | 你说 | 效果 |
|:----|:-----|:-----|
| 问推荐 | `今天吃什么` | 轻量模式，快速出三餐建议 |
| 吃完反馈 | `我吃了青椒炒牛肉+米饭` | AI 评估蛋白/蔬菜/油腻度 |
| 运动记录 | `周一：晨跑5km，30min，300kcal` | 联动调摄入 |

### 第三步：周总结

| 场景 | 你说 | 效果 |
|:----|:-----|:-----|
| 饮食分析 | `这周我吃了：（7天数据）` | 6维雷达评分 + 食堂替代方案 |
| 口味画像 | `看看我的口味` | 量化分析饮食风格 |
| 生成报告 | `生成全面健康档案` | 4 Tab 骚包 HTML |

### 快捷键

| 关键词 | 作用 |
|:-------|:-----|
| `预算不够` | 切换 ¥25~30/天 省钱模式 |
| `下周推荐` | 基于历史+口味+新菜出下周计划 |
| `一周运动记录` | 记录并分析周运动数据 |

### 用户定制

打开 SKILL.md，搜索 **`用户定制区`**，修改：
- 输出目录（默认 D:\wuliziliao\）
- 默认预算
- HTML 风格
- 示例档案

### 常见问题

**Q: 数据不够会怎样？**
A: <5天不生成雷达，提示继续记录。

**Q: 输错了怎么办？**
A: 发送纠正信息即可，AI 会更新 Memory。

**Q: 怎么换人用？**
A: 搜索 SKILL.md 的"用户定制区"，改示例档案和路径即可。
